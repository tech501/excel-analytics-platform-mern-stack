# Excel Analytics Frontend

A modern React application for analyzing Excel data with interactive 2D and 3D charts.

## Features

### 📊 Chart Types
- **2D Charts**: Bar charts, Line charts, Pie charts
- **3D Charts**: 3D Bar charts, 3D Scatter plots, 3D Surface plots
- **Interactive Controls**: Rotate, zoom, and pan 3D charts with mouse
- **Real-time Data**: Upload Excel files and generate charts instantly

### 🎯 Key Capabilities
- Excel file upload and parsing
- Multiple chart type selection
- Axis configuration (X, Y, Z for 3D)
- Demo data for testing
- Download charts as PNG/PDF
- Responsive design with modern UI
